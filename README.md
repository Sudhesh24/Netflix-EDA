Netflix Movies & TV Shows – Exploratory Data Analysis

Objective
To analyze the Netflix Movies and TV Shows dataset using Python to understand content distribution, genre trends, release patterns, and country-wise production insights.

Tools Used
Python
Pandas
Matplotlib
Seaborn

Key Insights
Netflix has more movies than TV shows.
Drama and International Movies are the most dominant genres.
Content production increased rapidly after 2016.
The United States is the top content-producing country, followed by India and the United Kingdom.

Files
netflix_eda.ipynb – Jupyter notebook with full exploratory data analysis
netflix_titles.csv – Dataset used for the analysis

Conclusion
this project demonstrates strong data analysis and visualization skills using real-world data.
